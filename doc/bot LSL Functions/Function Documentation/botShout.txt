Function: botShout(key botID, integer channel, string message);

The bot that has the ID "botID" will shout the message "message" on channel "channel".

botID - ID of the bot that will be shouting.
channel - output chat channel, any integer value
message - message to be transmitted

Example: bot Chat.lsl
